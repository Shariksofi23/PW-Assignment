1. Justify-content:

Role:Defines how flex items are distributed along the main axis of the flex container.
Usage in Flexbox: Allows adjustment of spacing between items or their alignment within the flex container. Values include `flex-start`, `flex-end`, `center`, `space-between`, and `space-around`, among others.

2. Align-items:

Role: Determines how flex items are aligned along the cross axis of the flex container.
Usage in Flexbox: Controls the positioning of items perpendicular to the main axis. Values include `flex-start`, `flex-end`, `center`, `baseline`, and `stretch`.

3. Gap:

Role:Sets the space between flex items or grid cells.
Usage in Flexbox/Grid:Provides a shorthand to define the gap between flex items in a flex container or grid cells in a grid container. It's an alternative to using margin or padding on individual items.

4. Flex-direction:

Role: Establishes the main axis direction of the flex container.
Usage in Flexbox: Defines whether flex items are laid out in rows (`row`), columns (`column`), reverse rows (`row-reverse`), or reverse columns (`column-reverse`).

5. Flex-wrap:

Role: Determines whether flex items are forced onto a single line or can wrap onto multiple lines.
Usage in Flexbox: Specifies if flex items should be wrapped when they exceed the container's width (`nowrap`, `wrap`, `wrap-reverse`).
