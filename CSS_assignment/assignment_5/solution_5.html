justify-items and justify-self are CSS properties used in CSS Grid Layout to align grid items within their grid container, but they work at different levels.

  1. justify-items: This property is used to align all the grid items within the grid container along the inline (row) axis.
  2. justify-self: This property is used to align a single grid item within its own grid cell along the inline (row) axis.

Here's an example to illustrate the difference between justify-items and justify-self:

HTML 

            < div class="grid-container">
                < div class="item item1">1 < /div >
                < div class="item item2">2< /div>

CSS 

            .grid-container {
                display: grid;
                grid-template-columns: repeat(3, 100px);
                grid-gap: 10px;
                justify-items: center; /* Aligns all grid items horizontally (inline axis) */
              }
              
              .item {
                background-color: #f0f0f0;
                border: 1px solid #ccc;
                padding: 20px;
              }
              
              .item1 {
                justify-self: start; /* Aligns this specific item to the start along the row axis */
              }
              
              .item2 {
                justify-self: center; /* Aligns this specific item to the center along the row axis */
              }
              
              .item3 {
                justify-self: end; /* Aligns this specific item to the end along the row axis */
              }

In this example:

  • .grid-container creates a grid layout with three columns each of 100px width and a 10px gap.
  • .grid-container creates a grid layout with three columns each of 100px width and a 10px gap.

However, for each individual item:

  • .item1 has justify-self: start;, which aligns it to the start (left) of its grid cell along the row axis.
  • .item2 has justify-self: center;, aligning it to the center of its grid cell along the row axis.
  • .item3 has justify-self: end;, aligning it to the end (right) of its grid cell along the row axis.

justify-items affects all grid items uniformly within the grid container,

justify-self allows you to individually control the alignment of specific items within their respective grid cells.
