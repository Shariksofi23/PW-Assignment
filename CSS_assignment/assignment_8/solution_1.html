var function

In CSS, the var() function is used to define and apply custom properties, allowing you to store and reuse values throughout your stylesheet. It enables the creation of variables (custom properties) that can hold various types of values such as colors, sizes, font stacks, or any other valid CSS value.

purpose and usage

  • Defining Variables: You can define a custom property using the `--` prefix followed by a name of your choice:` Here, `--primary-color` and `--secondary-color` are custom properties storing a color value. The `:root` selector defines these variables globally, making them accessible anywhere within the document.
  • Using Variables: Once you've defined variables, you can use them by referencing their names inside the `var()` function: In this example, the `#primaryBtn and #secondaryBtn` I'd uses the `var()` function to apply the stored color.
  • Dynamic Values and Cascade: Variables are dynamic and can be changed or overridden later in the stylesheet or within specific elements. They follow the normal cascading rules of CSS, allowing for easy customization and maintenance.
  • Fallback Values: It's also possible to provide fallback values in case the variable is not defined or not supported in a particular browser:

Reusable and Maintainable Code: The primary purpose of the `var()` function is to make CSS more maintainable and reusable by enabling the storage of values in variables. This allows for easier updates and consistency throughout the stylesheet by changing values in one place (the variable definition) instead of finding and modifying multiple instances of the same value.
