Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML documents?

Comments in HTML serve the purpose of adding notes or annotations within the code that are not visible in the rendered output of the webpage. They are useful for developers to document the code, provide explanations, or temporarily disable certain parts without deleting them.

Comments are enclosed between "".
They can be single-line or multi-line.
You can use comments to temporarily disable certain parts of the code, like the paragraph that's commented out.
Comments are useful for indicating sections of the code, such as marking the beginning or end of specific sections like the header or footer.
Remember that comments in HTML are for developers and are not visible to users viewing the webpage in a browser. They are essential for code readability, understanding, and maintenance.
