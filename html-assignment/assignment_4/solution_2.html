What are the Entities of html5?

HTML entities are special codes that represent characters that have a reserved meaning in HTML. They are used to display characters that would otherwise be interpreted as code or formatting

list out 5 commonly used html5 entities.

Here are five commonly used HTML entities in HTML5:

1. **<** - Less than sign (`<`): This entity is used to display the less than symbol `<` in HTML. It's commonly used when writing code examples or displaying text that includes the less than symbol without being interpreted as an opening HTML tag. br 2. **>** - Greater than sign (`>`): Similar to the previous entity, this is used to display the greater than symbol `>` in HTML without it being interpreted as an HTML tag closing.
3. **&** - Ampersand (`&`): This entity represents the ampersand symbol `&` and is used to display it in HTML without it being interpreted as the beginning of an entity.
4. **"** - Double quotation mark (`"`): It represents the double quotation mark symbol `"` and is often used when including attributes or displaying text containing double quotes inside HTML attributes.
5. **'** - Single quotation mark/apostrophe (`'`): This entity represents the single quotation mark or apostrophe `'`. It's used to display this character in HTML when needed, especially within attributes that already use double quotation marks.
