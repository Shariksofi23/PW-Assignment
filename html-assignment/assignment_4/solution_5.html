Exploring HTML5 Features

Feature 1: <canvas>

HTML5 introduced the <canvas> element, which provides a way to draw graphics, animations, and visualizations on a web page using JavaScript.

Feature 2: <video> and <audio>

Feature 3: Geolocation API

HTML5 introduced the Geolocation API, allowing websites to access a user's geographical location. However, for security reasons, access to the user's location requires their permission.
