1. html

    Primary Use Case Key Features/Benefits
    Building the structure of webpages
    • simple and easy to learn
    • compatible with all the web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Case Key Features/Benefits
    Controlling the visual appearance of web pages.
    • Separates content from design for easy maintenance.
    • Enables responsiveness across various devices.
    • Offers precise control over styling elements.

  3. JavaScript

    Primary Use Case Key Features/Benefits
    Adding interactivity and functionality to web pages.
    • Enables dynamic content updates without page reloads.
    • Supports event handling for user interactions.
    • Facilitates manipulation of HTML and CSS for dynamic changes.
